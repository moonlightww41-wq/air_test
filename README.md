# 運賃検索（複数旅程対応）

## 使い方
1. `index.html` を開く（Netlify/GitHub Pages でもOK）
2. 旅程を入力
   - テキスト貼り付け例:
     - `2025-08-21 東京→沖縄`
     - `8/22 沖縄→宮古`
   - またはフォームで旅程を追加

## データ更新（重要）
- データは `data/fare_source.tsv` を読み込みます。
- **このファイルを差し替えるだけで更新できます（アプリの修正不要）**。
- 文字化け回避のため、TSV/CSVは **UTF-8 (BOMあり推奨)**。

### 必須列（ヘッダ名）
- 搭乗期間開始
- 搭乗期間終了
- 出発地
- 到着地
- 価格タイプ（例: ピーク / 通常）
- 運賃（数値）
- 価格適用期間（例: 2025-07-01〜2025-08-31 / 2025-09-01〜2025-10-25）

※ `価格適用期間` が入っている場合、そこを分割して適用期間として扱います（全体期間との交差で自動補正）。

## 同義語（地名表記ゆれ）
- `data/place_aliases.csv` に `alias,canonical` を追加してください。
  - 例: `羽田,東京` / `那覇,沖縄`
